# P vs. NP: The $1 Million Question

The P vs. NP problem is a major unsolved problem in theoretical computer science. It deals with the speed at which a computer can solve a problem.

### üß© Defining the Classes

1. **P (Polynomial Time):** Problems that are "easy" for a computer to solve. 
   * *Example:* Sorting a list of numbers or searching for a name in a phone book.
2. **NP (Nondeterministic Polynomial Time):** Problems where a solution can be "easily" verified, but finding the solution might take billions of years.
   * *Example:* Solving a Sudoku puzzle. Checking a finished grid is fast, but solving an empty one is much harder.
