## 📚 Topics

* [Computer science fundamentals](./Computer%20science%20fundamentals) - Core theory like complexity, computability, and classic problem spaces.

Each folder contains short, focused notes designed to be read quickly and linked together over time.

## 🛠 How to Contribute (to myself)
1. Pick the best matching topic folder.
2. Add a Markdown file with a descriptive name, ideally using kebab-case.
3. Keep the note short, specific, and easy to scan.
4. Link it from related notes when the topic overlaps.
5. Commit and push!
