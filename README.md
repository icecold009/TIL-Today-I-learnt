## 🛠 How to Contribute (to myself)
1. Pick the best matching topic folder.
2. Add a Markdown file with a descriptive name, ideally using kebab-case.
3. Keep the note short, specific, and easy to scan.
4. Link it from related notes when the topic overlaps.
5. Start from [TIL_TEMPLATE.md](./TIL_TEMPLATE.md) when you want a consistent format.
6. Commit and push!
